# Project Management System

## 📋 Project Overview
A collaborative project management platform where users can create teams, manage projects, assign tasks, and track progress with real-time updates and analytics.

---

## 🎯 Core Features

### 1. Authentication & User Management
- User registration and login (JWT-based)
- Email verification
- Role-based access control (Admin, Team Lead, Member)

### 2. Team Management
- Create/edit/delete teams
- Invite members via email
- Accept/decline team invitations
- Team member roles and permissions
- Remove members from teams

### 3. Project Management
- Create projects within teams
- Project details (name, description, deadlines, status)
- Assign project managers
- Archive/restore projects
- Project progress tracking

### 4. Task Management
- Create/edit/delete tasks
- Task properties:
  - Title and description
  - Priority (Low, Medium, High, Urgent)
  - Status (To Do, In Progress, In Review, Done)
  - Due date and time
  - Assignee(s)
  - Tags/labels
- Subtasks/checklists
- Task dependencies
- File attachments
- Task comments and activity log

### 5. Dashboard & Analytics
- Personal dashboard showing assigned tasks
- Team overview with task distribution
- Progress charts (tasks completed over time)
- Deadline tracking and overdue alerts
- Productivity metrics per team member
- Burndown charts for projects

### 6. Notifications
- In-app notifications
- Email notifications for:
  - Task assignments
  - Approaching deadlines
  - Comments/mentions
  - Team invitations
  - Status changes

### 7. Search & Filters
- Search tasks across projects
- Filter by: assignee, status, priority, date range, tags
- Saved filter views
